🔥 Firebase Configuration (Permanent Storage)
Currently: Demo Mode (data resets on refresh).
To save data permanently, connect Firebase Firestore:
1. Go to console.firebase.google.com → Add Project → "smart-medical-shop"
2. Firestore Database → Create database → Start in test mode
3. Project Settings → Web App → Register → Copy config below
4. Also enable Hosting for free public URL (firebase deploy)

📤 GitHub & Deployment Guide
To upload to GitHub and run anywhere:
1. Create account at github.com
2. New Repository → "SmartMedicalShop" → Public
3. Upload SmartMedicalShop_FINAL.html → rename to index.html
4. Settings → Pages → Branch: main → Save
5. Your live URL: https://YOURNAME.github.io/SmartMedicalShop

For permanent data storage (Firebase Hosting):
npm install -g firebase-tools
firebase login → firebase init hosting → firebase deploy
Live URL: https://smart-medical-shop.web.app
